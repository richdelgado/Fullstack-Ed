LINK A LA CLASE
  • Posicionarse en la carpeta en tu computadora donde crearas el repositorio local (Git) utilizando la terminal
  • ingresar el comando "git init"
  • Puedes corroborar si se creo la carpeta oculta con el comando "ls -al" y mostrara los archivos incluida la nueva carpeta .git
  • Creas tu archivo .html, lo editas y guardas (computadora) en la carpeta donde se creo el repositorio
  • En la terminal ingresamos el comando "git status" para ver a que branch estamos apuntando
  • En la terminal ingresamos el comando "git add +nombbre del archivo" o podemos usar el comando "git add ." y agrega todos los archivos en esa carpeta
  • Podemos ingresar el comando "git status" para ver los cambios y confirmar que aun no hay commits
  • Si deseas remover el archivo de STAGED usamos el comando "git rm + nombre del archivo" y podemos corroborar con "git status" pero sigue en memoria RAM
  • Para eliminarlo de la memoria RAM usamos "rm --cached + nombre del archivo" y podemos corroborar con "git status"

Nota: STAGED es como un limbo, no se ha hecho commit pero esta en memoria RAM un por lo que necesitamos indicarle a la consola que hacer con el archivo

Nota: Siempre agregar comentarios cuando hacemos commits (es una buena practica)
